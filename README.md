# Cloth Shop E-Commerce Website

A modern full-stack clothing e-commerce web application built using Node.js, Express, and MongoDB.

## Features

- User Login and Authentication
- Admin Dashboard
- Product Management
- Shopping Cart
- Wishlist System
- Responsive Design
- Modern UI/UX

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Installation

1. Clone the repository

git clone https://github.com/JuliusCeasar777/cloth-shop.git

2. Install dependencies

npm install

3. Run the server

node server.js
